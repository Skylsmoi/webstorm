## Setup
### Node js
```
curl -sL https://deb.nodesource.com/setup_7.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### Get the source
```
git clone ssh://git@gitlab.algoo.fr:10022/apc/apcfrontend.git
```

### install all dependencies
```
npm install
```

## Build
### Build all sources
```
npm run build
```
