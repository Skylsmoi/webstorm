## Setup
### Node js
```
curl -sL https://deb.nodesource.com/setup_7.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### Get the source
```
git clone git@github.com:Skylsmoi/webstorm.git && cd webstorm
```

### install all dependencies
```
npm install
```

## Build
### Start dev serv
```
npm run servdev
```
then open http://localhost:8083/
## OR
### Build all sources
```
npm run build
```
